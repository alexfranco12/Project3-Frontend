# GA-SEI-Project-Purlieu

Purlieu is a local search and discovery app that provides recommendations of places to go nearby. 

### Technologies used

* Javascript
* React
  - React Routers
  - React Select
  - React Bootstrap
* Third party API's:
  - Google Maps
  - Google Places
* Flexbox
* Heroku
* Express
* MongoDB
* Mongoose
